Layout

Web sitesi genellikle header,nev,content,footer ortak alanları tek bir yerde tasarladığımız şeydir

            header: Web sitesini ne üstnde yer alan başlık, genellikle logo veya website adı içerir
            Nav: Gezintiçubuğu ziyaretçilerin ewb sitenizde gezinmesine yardımcı olacak bağlantıların bir listesini içerir
            Content: Düzen, genellikle hedef kullanıcılara bağlıdır.
            footer: Alt bilgi safamızın en altında yer alır. Genellikle telif hakkı bilgileri içeren kısımdır.
